# 📊 Visit with Us - MLOps Project

## 🚀 Project Overview
"Visit with Us" is a machine learning project designed to predict whether a customer will purchase a **Wellness Tourism Package**. The goal is to help businesses improve marketing efficiency by targeting high-potential customers.

---

## 🎯 Business Objective
- Increase conversion rates
- Reduce marketing costs
- Improve customer targeting using data-driven insights

---

## 🧠 Machine Learning Objective
Build a **binary classification model** to predict:

- `0` → Customer will NOT purchase  
- `1` → Customer WILL purchase  

---

## ⚙️ Tech Stack

## 📊 Model Details
- Algorithm Used: **Random Forest Classifier**
- Includes:
  - Data preprocessing (scaling + encoding)
  - Pipeline-based model training
- Evaluation Metrics:
  - Accuracy
  - Precision
  - Recall
  - F1 Score

---

## 📈 Key Insights
- High-income customers are more likely to purchase
- Passport holders show higher travel intent
- Follow-ups increase conversion rates
- Customer satisfaction strongly impacts buying decisions

---

## 💡 Business Recommendations
1. Target high-income segments
2. Focus on customers with travel history
3. Improve sales pitch quality
4. Increase follow-up interactions
5. Use ML predictions instead of manual targeting
